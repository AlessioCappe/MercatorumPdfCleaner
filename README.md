# MercatorumPdfCleaner
Pulitore per le dispense .pdf di mercatorum <br>
<br>
Rimuove le immagini di watermark e copre la parte relativa al copyright per avere maggiore leggibilita e più spazio per prendere appunti
<br>
## Librerie necessarie
- [PyMuPDF](https://pymupdf.readthedocs.io/en/latest/)
- [Tesseract](https://tesseract-ocr.github.io/) (necessaio installare il pacchetto dati per la lingua italiana)
